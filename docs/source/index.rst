OH Auto Statistical
===================

Fully automated flood estimation for UK catchments using the latest revisions to the Flood Estimation Handbook
Statistical Method as developed by `Kjeldsen, Jones and Bayliss (2008) <https://www.gov.uk/government/publications/improving-the-flood-estimation-handbook-feh-statistical-procedures-for-flood-frequency-estimation>`_.
